deebait
A website made for 1-to-1 anonymous chat experience but with a twist: You get matched up with someone of an opposite preference as you. If you chose that you're a cat lover beforehand, it could match you with a dog lover instead.
@Github @Deebait
artgrid
A set of python scripts meant to be used sequentially allowing its users to create a stitched up collage forming the likeness of an image by having it made of a large collection of smaller images.
@Github
fairu
An unhosted website made for indexing academic files. Complete with a search functionality and dashboard for administrators to verify file authenticity.
@Github
paramour
A matchmaking website with no database. Instead, it generates a QR code containing compressed and encoded user data, meant to be used with another QR code from another user. It can then compute compatibility based on the set metrics.
@Github @Paramour
aspire-ph
In collaboration with an amazing team from De La Salle University, a web application that allows users to easily estimate the chances of getting infected through a user-friendly and intuitive interface. It also has language support especially for Filipinos.
@Github @AspirePH
